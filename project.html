Notetaking Web App

GT Scribe was created with the intent to facilitate ease-of-use for a notetaking service provided to students with disabilities by the Office of Disability Services at Georgia Tech. The technology stack for the back-end was developed using Flask, Python, SQLite, and Amazon S3. The front-end was developed using HTML5, Jinja, CSS3, Bootstrap, JavaScript and jQuery. The source code for this project can be accessed here.

Fatal Police Shootings Visualization

This information visualization was created to allow for an interactive display of data collected from the Washington Post about fatal police shootings that have occurred in the United States since January 2015, which can be found here. The visualization was created using HTML5, CSS3, D3, and Javascript. The visualization can be filtered by gender, race, age, and whether or not the police officer involved was wearing a body camera. The source code for this project can be accessed here and the final project can be viewed here with a video demo that can be watched here.
